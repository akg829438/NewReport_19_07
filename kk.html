**Global State:** Manage global state using Context API for user authentication, profile data, etc. - **Local State:** Use local state within components where necessary. #### Task Description 1. **User Login and Redirection to Edit Profile** - **Task:** Implement login functionality. - **Description:** On successful login, check if the user is new. If so, redirect to the edit-profile page. After updating the profile, redirect to the home page. 2. **Home Page with Tabs for Astrologer Services** - **Task:** Implement tabbed navigation on the home page. - **Description:** Create three tabs: Chat with Astrologers, Call with Astrologers, Video Call with Astrologers. Ensure the Chat tab is selected by default. Fetch and display astrologers based on the selected tab. 3. **Astrologer Listing** - **Task:** Create components to list astrologers. - **Description:** Display astrologers based on the selected service (chat, call, video call). Fetch data from the backend and render it in a list or grid format. 4. **Astrologer Filter By State/Language** - **Task:** Allow users to filter astrologers by language or state. 5. **Astrologer Filter By Expertise** - **Task:** Allow users to filter astrologers by expertise. 6. **Astrologers Search Functionality** - **Task:** Enable users to search for astrologers. 7. **Filter By Category** - **Task:** Allow users to filter astrologers by category. 8. **Chat With Astrologers** - **Task:** Enable users to chat with astrologers. 9. **Call With Astrologers** - **Task:** Enable users to call astrologers. 10. **Video Call With Astrologers** - **Task:** Enable users to video call astrologers. 11. **Chat Assistant** - **Task:** Allow users to join a chat assistant session if they have had a paid session before. Users can only have 5 chat sessions per day. 12. **Emergency Voice Call** - **Task:** Allow users to join an emergency voice call if they have had a paid session before. Emergency calls will deduct more from the account than normal calls, according to the emergency rate. 13. **Emergency Video Call** - **Task:** Allow users to join an emergency video call if they have had a paid session before. Emergency calls will deduct more from the account than normal calls, according to the emergency rate. 14. **Live Streaming** - **Task:** Allow users to join any live session if an astrologer has started streaming.
